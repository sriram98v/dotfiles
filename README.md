# Dotfiles
Here are my personal desktop setup on Arch linux** with **i3-gaps** as window manager.

## Desktop
![setup](/.config/desktop.jpg)

## Requirements
* **Window manager** = i3-gaps

* **Wallpaper tool** = feh

* **Bar** = polybar

* **Compositor** =  ibhagwan-picom

* **Application Menu** = rofi

* **Terminal** = urxvt
